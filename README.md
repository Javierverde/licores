Licores Artesanales - Página Web

Bienvenido al repositorio de la página web de Ikigai Licores Artesanales. Este proyecto es una representación digital de nuestra pasión por los licores hechos a mano, donde los visitantes pueden explorar nuestra gama de productos, conocer más sobre nuestros eventos y descubrir la historia detrás de nuestra marca.


<br>Descripción:

<br>Esta página web está diseñada para presentar de manera atractiva y funcional los licores artesanales que ofrecemos. Se compone de cinco secciones principales:
<br> -Inicio: Una introducción a la marca con un diseño visual impactante. 
<br> -Productos: Un catálogo de nuestros licores artesanales, con detalles de cada producto.
<br> -Eventos: Información sobre eventos próximos y pasados donde se podrán degustar y adquirir nuestros productos.
<br> -Nosotros: Una sección dedicada a contar la historia de nuestra marca y nuestro equipo.
<br> -Contacto: Formulario y detalles de contacto para que los usuarios puedan comunicarse con nosotros.


<br>Características:

<br> -Responsive Design: La página está diseñada para ser completamente responsiva, asegurando una experiencia óptima en dispositivos móviles, tabletas y escritorios.
<br> -Navegación Intuitiva: Una barra de navegación clara y sencilla permite a los usuarios moverse fácilmente entre las diferentes secciones.
<br> -Catálogo Dinámico: La sección de productos se presenta en un formato visual atractivo y fácil de navegar.
<br> -Integración con Eventos: Los usuarios pueden mantenerse al tanto de nuestros eventos y actividades directamente desde la página.
<br> -Formulario de Contacto: Un formulario de contacto funcional para facilitar la comunicación con los visitantes.


<br>Tecnologías Utilizadas:

<br>-HTML5: Para la estructura y el contenido de las páginas.
<br>-CSS3: Para el diseño visual y la disposición de los elementos.
<br>-Sass/SCSS: Preprocesador de CSS utilizado para una mejor organización y mantenimiento de los estilos.
<br>-Bootstrap: Framework de CSS utilizado para facilitar el diseño responsivo.
<br>-GitHub Pages: Para el despliegue y alojamiento de la página.
